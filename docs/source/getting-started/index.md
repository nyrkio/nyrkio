# Getting Started

Welcome to Nyrkiö! This section will help you get started with performance change detection.

## What is Nyrkiö?

Nyrkiö is an open-source platform for continuous performance engineering that automatically detects performance regressions and improvements in your test data.

## Key Benefits

- **Early Detection**: Find performance issues before they reach production
- **Accurate**: Detect changes as small as 0.5-1%
- **Automated**: Integrates with your CI/CD pipeline
- **Collaborative**: Share results with your team
