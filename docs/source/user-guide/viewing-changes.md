# Viewing Changes

## Dashboard

The main dashboard shows:
- List of all your tests
- Recent change points
- Trend graphs with annotations

## Change Point Details

Click on any change point to see:
- Affected metrics
- Magnitude of change
- Statistical significance (p-value)
- Git commit information
- Before/after values

## Filters

Filter changes by:
- Time range
- Repository
- Branch
- Test name
- Metric name

## Graphs

Interactive time-series graphs show:
- Historical performance data
- Detected change points (marked with vertical lines)
- Confidence intervals
- Trend lines

## Export

Export data as:
- CSV
- JSON
- PNG (graph screenshots)
